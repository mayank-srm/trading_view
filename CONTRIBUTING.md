# Contributing

Thanks for helping improve these TradingView tools.

## Guidelines

- Keep Pine Script changes readable and documented where the logic is not obvious.
- Avoid repainting behavior unless it is intentional and clearly explained.
- Update the relevant Markdown guide when inputs, alert names, dashboard states, or defaults change.
- Include a chart screenshot or image when changing visual output.
- Keep this project focused on decision support, not automated trade execution.

## Pull Request Checklist

- The script compiles in TradingView.
- The behavior has been checked on standard candles.
- New inputs have sensible defaults.
- Risk or limitation notes are updated when needed.
