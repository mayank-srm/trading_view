# Instagram Reel Video

Remotion composition for a vertical Instagram Reel/Story promoting the automated TradingView push workflow.

## Render

```bash
npm install
npm run render
```

Output:

```text
out/tradingview-auto-reel.mp4
```

## Preview

```bash
npm run studio
```

## Specs

- Aspect ratio: 9:16
- Resolution: 1080 x 1920
- Duration: 15 seconds
- Frame rate: 30 fps
